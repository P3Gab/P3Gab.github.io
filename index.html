Задача

Да се добави запис към host файла (ime.p3), който да сочи към вашата локална машина (127.0.0.1)

Задача

Да се проверят MX, NS, A, TXT записите за students.tugab.bg

Създаване на адреси към вашите колеги

Проверете ip адресите на вашите колеги и създайте запис в host файла, който да сочи към машината на вашия колега!

Инсталиране и тестване ssh връзка

команди ssh, nestat (проверка отворени портове)

Домашно

  • Инсталиране docker и проверка инсталацията,
  • Добавяне на локалния потребител група docker (sudo usermod -a -G docker user_name)
  • Конфигуриране връзка с ключ (без парола) към виртуалната машина
  • Добавяне на име към host файла, което да сочи към адреса на виртуалната машина

Ден 2 / Dockerfile

Създаване на private repository в github

  • Създаване на файл index.html с произволно съдържание
  • Създаване на Dockerfile със следното съдържание
    • FROM debian:latest
RUN apt-get update
RUN apt-get install -y dnsutils net-tools \
  openssh-server telnet inetutils-ping \
  vim nano less dnsutils tcpdump apache2 
LABEL version="1.1"
LABEL description="none"
EXPOSE 22
EXPOSE 80/tcp
COPY index.html /var/www/html/index.html
CMD ["/usr/sbin/apachectl", "-D", "FOREGROUND"]
